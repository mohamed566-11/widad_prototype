# 🌸 تطبيق وداد تك (Widad Tech) - الدليل الشامل

هذا المستودع يحتوي على الواجهة الأمامية لمشروع "وداد تك"، المنصة الرقمية المتكاملة لصحة المرأة.
المشروع مبني باستخدام **React**, **Vite**, **TypeScript**, و **Tailwind CSS**.

## 🛠️ المتطلبات الأساسية (Prerequisites)

قبل البدء، تأكد من تثبيت البرامج التالية على جهازك:

- **Node.js** (إصدار 18 أو أحدث) - [للتحميل من هنا](https://nodejs.org/en/)
- **pnpm** (مدير الحزم المعتمد في المشروع) - الأسرع والأنسب.
  - _لتثبيته افتح الطرفية (Terminal) واكتب:_ `npm install -g pnpm`

---

## 🚀 خطوات التثبيت والتشغيل (Installation & Setup)

1. **نسخ المشروع إلى جهازك (Clone repository):**

   ```bash
   git clone <رابط-مستودع-جيت-هب-الخاص-بك>
   cd widad-replit2-main
   ```

2. **الدخول إلى مسار الواجهة الأمامية الأساسي:**

   ```bash
   cd artifacts/widad
   ```

3. **تثبيت الحزم (Install packages):**

   ```bash
   pnpm install
   ```

4. **إعداد متغيرات البيئة (Env variables):**
   - قم بإنشاء ملف `.env.local` داخل المسار `artifacts/widad/`
   - ضع فيه مفتاح الـ API الخاص بـ Gemini (لو تواجد) كالتالي:

   ```env
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

   _(إن لم يتوفر، فالتطبيق مبرمج للعمل بالبيانات الافتراضية محلياً ولن يتعطل)_.

5. **شغّل المشروع في بيئة التطوير:**
   ```bash
   pnpm run dev
   ```
   سيعمل المشروع على الرابط `http://localhost:5173`.

## 📁 الهيكل العام للمشروع (Folder Structure)

مسارنا الأساسي هو للعمل كفريق واجهة أمامية سيكون داخل **`artifacts/widad/src/`**:

```plaintext
src/
├── components/      # المكونات المشتركة الدائمة (الأزرار، التخطيطات، القوائم الجانبية)
├── pages/           # مسارات الصفحات الرئيسية، وتتفرع إلى:
│   ├── public/      # الصفحات العامة (الرئيسية، مقالات، أطباء عامة)
│   ├── auth/        # شاشات الدخول واختيار الصلاحيات
│   ├── patient/     # لوحة تحكم المريضة ومتتبعاتها
│   ├── doctor/      # لوحة تحكم الطبيبة ومواعيدها
│   └── admin/       # لوحة تحكم الإدارة الكاملة
├── store/           # إدارة حالة التطبيق محلياً (مثل Zustand لتسجيل الدخول)
├── mock/            # داتا وهمية (Mock Data) جاهزة للتبديل لاحقاً مع الـ Backend
├── lib/             # ملفات التكوين (Tailwind Utils، دوال مساعدة، وضبط الذكاء الاصطناعي)
├── App.tsx          # موجّه الروابط الأساسي (Routes Mapping)
└── index.css        # نظام تصميم الألوان للـ Tailwind
```

## 🎨 ملاحظات التصميم (Design System)

- **الخطوط:** تم وضع خط `Tajawal` ليكون هو الخط السلس في ملف التكوينات `tailwind.config.cjs`.
- **الألوان:** جميع الألوان تم ضبطها وتوحيدها كمتغيرات (CSS variables) داخل `index.css`. للوصول للون التطبيق الرئيسي، استخدم `bg-primary` و `text-primary`.
- **تخطيط الصفحات (Layouts):**
  1. لا تكرر الكود في الصفحات العامة: اجعل الصفحة ابناً للمكون `<PublicContentLayout>` وسيرث الناف بار والفوتر أوتوماتيكياً.
  2. في صفحات الداشبورد (أي لوحة تحكم): اغلف المحتوى بمكون `<SidebarLayout>` وسيرث شكل اللوحة الجانبية الفخمة.

بالتوفيق لفريق التطوير! 🚀
